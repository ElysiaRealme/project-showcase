# Project Showcase / 项目展示

This repository contains sanitized project READMEs and case studies for private projects whose source code cannot be published due to client Non-Disclosure Agreements (NDAs).

本仓库用于存放因客户保密协议（NDA）无法公开源码的项目说明、架构笔记与脱敏案例总结。

## How to Read / 如何浏览

- Start from the category index below.
- Each project should have its own folder and `README.md`.
- Sensitive information must be removed before publishing: client names, credentials, private endpoints, internal screenshots, production data, and unreleased business logic.

- 从下面的分类索引开始浏览。
- 每个项目建议独立一个目录，并在目录内放置 `README.md`。
- 发布前必须移除敏感信息：客户名称、凭据、私有接口、内部截图、生产数据、未公开业务逻辑等。

## Disclosure Policy / 公开边界

Source code is private by default. This repository is for portfolio review, interview discussion, and self-management only.

源码默认不公开。本仓库仅用于作品集展示、面试沟通与个人项目管理。
